How it works
Three steps to agent-native software
01 — Discover

Search by capability

Query the registry by what you need to do, not what tool to use. "resize an image" returns the right CLI package with its full capability manifest.

02 — Install

One command setup

Every package follows a unified standard. Install via pip, npm, or brew. Dependencies are declared in the manifest and resolved automatically.

03 — Use

Agent runs it

Your agent reads the manifest, constructs the right command, and gets structured JSON back. No GUI, no brittle screen scraping, no API fragmentation.

Why CLI
The universal agent interface

GUI automation is brittle. APIs are fragmented. CLI is the interface that already works everywhere.

Deterministic

Same input, same output. No pixel coordinates, no timing issues, no race conditions. Agents get reliable, reproducible results every time.

Cross-platform

CLI works on Linux, macOS, and Windows. One interface definition, universal compatibility. No platform-specific GUI automation scripts.

Composable

Pipe outputs between tools. Chain operations. Build complex workflows from simple, focused commands. Unix philosophy, agent-ready.

Self-describing

Every command has --help. Every package has a manifest. Agents can introspect capabilities at runtime without external documentation.

Standard
The CLI manifest spec

A single JSON file that tells any agent what a CLI can do.

Machine-readable by default

Every package ships a manifest.json describing capabilities, input/output types, platform support, and quality metrics. Agents parse it at install time.

  • Semantic capability descriptors
  • Typed argument and output schemas
  • Dependency and platform declarations
  • Test count and quality score
  • Versioned with changelogs
  • Usage examples for agent context
manifest.jsonJSON
{ "name": "cli-anything-gimp", "version": "2.1.0", "description": "Full GIMP image editing via CLI", "capabilities": [ "image.open", "image.resize", "image.filter", "image.export", "layer.create", "layer.merge" ], "input_formats": ["png", "jpg", "psd"], "output": "json", "platform": ["linux", "macos", "windows"], "install": "pip install cli-anything-gimp", "requires": ["gimp>=2.10"], "tests_passed": 187, "quality_score": 0.96 }
