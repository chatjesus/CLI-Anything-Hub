How it works
From software to agent in 60 seconds
01

Discover by capability

Search the registry by what you need to do, not what tool to use. Every package has a machine-readable manifest.

02

Install with pip

Every CLI tool is a pip package. Install once, use forever. Dependencies are declared and resolved automatically.

03

Call from your agent

Your agent runs the CLI command, gets structured JSON back. No API wrappers, no SDK versions, no auth complexity.

04

Compose workflows

Chain multiple tools together. Resize an image with GIMP, upload to Slack, log to Notion — all in one agent workflow.
